## Получить список кандидатов

```http
GET /v1/candidates
```

### Параметры запроса

| Параметр | Тип | Описание |
|----------|-----|----------|
| `vacancy_id` | string | Фильтр по вакансии |
| `stage` | string | Фильтр по этапу воронки |
| `min_score` | integer | Минимальная оценка AI |
| `limit` | integer | Количество (макс. 100) |
| `offset` | integer | Смещение для пагинации |

### Пример запроса

```bash
curl -X GET "https://api.qbs-autonaim.ru/v1/candidates?vacancy_id=vac_123&min_score=70&limit=20" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## Изменить этап кандидата

```http
PATCH /v1/candidates/{candidate_id}
```

### Тело запроса

```json
{
  "stage": "offer"
}
```

### Пример

```bash
curl -X PATCH "https://api.qbs-autonaim.ru/v1/candidates/cand_abc123" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"stage": "offer"}'
```
