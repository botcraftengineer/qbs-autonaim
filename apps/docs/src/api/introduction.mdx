## Обзор

REST API позволяет программно взаимодействовать с QBS Автонайм:

- Создавать и управлять вакансиями
- Получать данные о кандидатах
- Изменять статусы и этапы
- Интегрироваться с внешними системами

## Базовый URL

```
https://api.qbs-autonaim.ru/v1
```

## Формат данных

- Все запросы и ответы в формате JSON
- Кодировка UTF-8
- Даты в формате ISO 8601 (`2025-01-14T10:30:00Z`)

## Версионирование

API версионируется через URL:

```
/v1/candidates  — текущая версия
/v2/candidates  — будущая версия (при появлении)
```

<Info>
  Мы поддерживаем предыдущие версии API минимум 12 месяцев после выхода новой версии.
</Info>

## Лимиты запросов

| Тариф | Лимит | Период |
|-------|-------|--------|
| Бесплатный | 100 запросов | в час |
| Стартап | 1,000 запросов | в час |
| Бизнес | 10,000 запросов | в час |
| Enterprise | Без лимита | — |

При превышении лимита вы получите ответ `429 Too Many Requests`.
