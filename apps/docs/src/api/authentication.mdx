## Получение API-ключа

<Steps>
  <Step title="Откройте настройки API">
    Перейдите в **Настройки → API → Ключи доступа**.
  </Step>
  <Step title="Создайте новый ключ">
    Нажмите **+ Создать ключ** и укажите название.
  </Step>
  <Step title="Скопируйте ключ">
    Ключ показывается только один раз. Сохраните его в надёжном месте.
  </Step>
</Steps>

<Warning>
  Никогда не делитесь API-ключом и не храните его в публичных репозиториях.
</Warning>

## Использование ключа

Передавайте ключ в заголовке `Authorization`:

```bash
curl -X GET "https://api.qbs-autonaim.ru/v1/candidates" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
```

## Области доступа (Scopes)

При создании ключа выберите разрешения:

| Scope | Описание |
|-------|----------|
| `candidates:read` | Чтение данных кандидатов |
| `candidates:write` | Создание и изменение кандидатов |
| `vacancies:read` | Чтение вакансий |
| `vacancies:write` | Управление вакансиями |
| `webhooks:manage` | Настройка webhooks |

## Ошибки аутентификации

| Код | Описание | Решение |
|-----|----------|---------|
| 401 | Ключ не передан или неверен | Проверьте заголовок Authorization |
| 403 | Недостаточно прав | Проверьте scopes ключа |
| 429 | Превышен лимит запросов | Подождите или повысьте тариф |
```
