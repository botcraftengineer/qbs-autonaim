## Получить список вакансий

```http
GET /v1/vacancies
```

### Параметры

| Параметр | Тип | Описание |
|----------|-----|----------|
| `status` | string | `active`, `closed`, `draft` |
| `limit` | integer | Количество (макс. 100) |
| `offset` | integer | Смещение |

### Пример ответа

```json
{
  "data": [
    {
      "id": "vac_xyz789",
      "title": "Менеджер по продажам",
      "status": "active",
      "location": "Москва",
      "salary_min": 150000,
      "salary_max": 200000,
      "candidates_count": 45,
      "created_at": "2025-01-01T10:00:00Z"
    }
  ]
}
```

## Создать вакансию

```http
POST /v1/vacancies
```

### Тело запроса

```json
{
  "title": "Frontend-разработчик",
  "description": "Описание вакансии...",
  "requirements": "Требования...",
  "location": "Удалённо",
  "salary_min": 180000,
  "salary_max": 250000,
  "employment_type": "full_time"
}
```

## Закрыть вакансию

```http
POST /v1/vacancies/{vacancy_id}/close
```

### Тело запроса

```json
{
  "reason": "hired",
  "hired_candidate_id": "cand_abc123"
}
```
