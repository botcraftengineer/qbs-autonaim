## Готовые отчёты

QBS Автонайм включает набор готовых отчётов для типовых задач:

### Операционные отчёты

| Отчёт | Описание | Период |
|-------|----------|--------|
| Статус вакансий | Все открытые вакансии с количеством кандидатов | Текущий |
| Воронка найма | Конверсия по этапам для каждой вакансии | Неделя/Месяц |
| Активность рекрутеров | Действия каждого члена команды | Неделя/Месяц |
| Источники кандидатов | Распределение откликов по каналам | Месяц |

### Аналитические отчёты

| Отчёт | Описание | Период |
|-------|----------|--------|
| Time-to-Hire | Среднее время закрытия по типам вакансий | Квартал |
| Качество найма | Прохождение испытательного срока | Квартал |
| ROI источников | Стоимость найма по каналам | Год |
| Причины отказов | Почему кандидаты не проходят | Месяц |

## Создание отчёта

<Steps>
  <Step title="Откройте конструктор">
    Перейдите в **Аналитика → Отчёты → Создать**.
  </Step>
  <Step title="Выберите метрики">
    Отметьте показатели, которые хотите видеть в отчёте.
  </Step>
  <Step title="Настройте фильтры">
    Укажите:
    - Период данных
    - Вакансии (все или конкретные)
    - Рекрутеры
    - Источники
  </Step>
  <Step title="Выберите визуализацию">
    - Таблица
    - Линейный график
    - Столбчатая диаграмма
    - Круговая диаграмма
  </Step>
  <Step title="Сохраните или экспортируйте">
    Сохраните отчёт для регулярного использования или выгрузите в Excel/PDF.
  </Step>
</Steps>

## Автоматические отчёты

Настройте отправку отчётов по расписанию:

<Tabs>
  <Tab title="Email">
    Получайте отчёт на почту:
    - Ежедневно (сводка за день)
    - Еженедельно (по понедельникам)
    - Ежемесячно (1 числа)
  </Tab>
  <Tab title="Telegram">
    Краткая сводка в Telegram-бот:
    - Утренняя сводка в 9:00
    - Вечерняя сводка в 18:00
    - По запросу
  </Tab>
  <Tab title="Slack/Teams">
    Отправка в рабочий чат:
    - В канал #hr-analytics
    - Личным сообщением
  </Tab>
</Tabs>

## Экспорт данных

Выгружайте данные в удобном формате:

- **Excel (.xlsx)** — для дальнейшего анализа
- **PDF** — для презентаций и отчётов руководству
- **CSV** — для импорта в другие системы
- **Google Sheets** — прямая интеграция

<Tip>
  При регулярной выгрузке используйте автоматические отчёты — они приходят в нужном формате без ручной работы.
</Tip>
