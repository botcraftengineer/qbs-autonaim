## Обзор

Раздел **Кандидаты** — это центр управления всеми откликами и соискателями в вашей системе. Здесь вы можете:

- Просматривать все отклики в одном месте
- Фильтровать кандидатов по вакансиям, статусам и оценкам
- Использовать AI для автоматического скрининга
- Перемещать кандидатов по этапам воронки
- Оставлять комментарии и делиться с командой

## Откуда приходят кандидаты

QBS Автонайм собирает кандидатов из множества источников:

| Источник | Описание |
|----------|----------|
| hh.ru | Отклики с HeadHunter через официальную интеграцию |
| SuperJob | Отклики через API SuperJob |
| Telegram | Отклики через Telegram-бота компании |
| Email | Резюме, отправленные на корпоративную почту |
| Реферальная программа | Кандидаты от сотрудников компании |
| Ручное добавление | Кандидаты, добавленные вручную рекрутером |

<Tip>
  Все кандидаты автоматически дедуплицируются. Если один человек откликнулся с разных площадок, система объединит его отклики в одну карточку.
</Tip>

## Карточка кандидата

Каждый кандидат имеет карточку со следующей информацией:

- **Основные данные:** ФИО, контакты, фото
- **Опыт работы:** История трудоустройства с датами и описанием
- **Навыки:** Технические и soft skills
- **Образование:** Вузы, курсы, сертификаты
- **AI-оценка:** Балл релевантности и обоснование
- **История взаимодействия:** Переписка, комментарии команды, этапы воронки
- **Документы:** Прикреплённые файлы (резюме, тестовые задания)

<Note>
  Все данные извлекаются автоматически из резюме. Вы можете редактировать информацию вручную при необходимости.
</Note>
