## Что такое воронка найма

**Воронка найма** — это визуальное представление всех этапов, через которые проходит кандидат от первого отклика до выхода на работу. В QBS Автонайм вы видите всех кандидатов на канбан-доске с колонками-этапами.

</Frame>

## Стандартные этапы воронки

По умолчанию воронка включает следующие этапы:

| Этап | Описание | Действия |
|------|----------|----------|
| **Новый** | Только что откликнулся | AI-скрининг, первичный отбор |
| **Скрининг** | Проходит отбор | Изучение резюме, телефонный скрининг |
| **Интервью** | Приглашён на собеседование | HR-интервью, техническое интервью |
| **Тестовое** | Выполняет тестовое задание | Оценка результатов |
| **Финал** | Финальный этап | Собеседование с руководителем, проверка рекомендаций |
| **Оффер** | Получил предложение | Согласование условий |
| **Нанят** | Принял оффер | Оформление документов |
| **Отказ** | Отклонён или отказался | Архив |

## Работа с воронкой

### Перемещение кандидатов

Перетащите карточку кандидата в нужную колонку, чтобы изменить его статус. При перемещении:

1. Статус обновляется мгновенно
2. Фиксируется время перехода
3. Отправляется автоответ (если настроен)
4. Команда получает уведомление (если настроено)

<Tip>
  Используйте горячие клавиши для быстрого перемещения:
  - `→` — следующий этап
  - `←` — предыдущий этап
  - `R` — отклонить
</Tip>

### Массовые действия

Выберите несколько кандидатов (Ctrl+клик или Shift+клик) и выполните действие для всех:

- Переместить на этап
- Отправить сообщение
- Добавить тег
- Отклонить с причиной

### Фильтры и сортировка

Используйте фильтры для быстрого поиска:

```
Фильтры:
• По вакансии
• По источнику (hh.ru, SuperJob, email)
• По дате отклика
• По оценке AI (80+, 60-79, и т.д.)
• По тегам
• По рекрутеру

Сортировка:
• По дате (новые/старые)
• По оценке (высокая/низкая)
• По времени на этапе
```

## Настройка воронки

Вы можете настроить воронку под процессы вашей компании.

### Добавление этапа

<Steps>
  <Step title="Откройте настройки воронки">
    Перейдите в **Настройки → Воронка найма**.
  </Step>
  <Step title="Нажмите «+ Добавить этап»">
    Кнопка находится в конце списка этапов.
  </Step>
  <Step title="Настройте этап">
    Укажите:
    - Название этапа
    - Цвет для визуального выделения
    - Автоответ при переходе (опционально)
    - Дедлайн нахождения на этапе (опционально)
  </Step>
  <Step title="Определите позицию">
    Перетащите этап в нужное место в последовательности.
  </Step>
</Steps>

### Удаление и редактирование

<Warning>
  При удалении этапа все кандидаты на нём будут перемещены на предыдущий этап. Убедитесь, что это не нарушит ваш процесс.
</Warning>

## Автоматизация воронки

Настройте автоматические действия для экономии времени:

<AccordionGroup>
  <Accordion title="Автоматическое продвижение">
    Кандидаты с оценкой 80+ автоматически переходят на этап «Скрининг» с отправкой приглашения.
  </Accordion>
  
  <Accordion title="Автоотказ">
    Кандидаты с оценкой ниже 40 автоматически получают вежливый отказ.
  </Accordion>
  
  <Accordion title="Напоминания">
    Если кандидат находится на этапе дольше установленного времени, вы получите напоминание.
  </Accordion>
  
  <Accordion title="Уведомления команде">
    При переходе кандидата на определённый этап ответственный получает уведомление.
  </Accordion>
</AccordionGroup>

## Аналитика воронки

Отслеживайте эффективность найма:

- **Конверсия между этапами** — сколько % кандидатов переходит дальше
- **Среднее время на этапе** — где возникают задержки
- **Причины отказов** — почему кандидаты отсеиваются
- **Сравнение по вакансиям** — какие вакансии закрываются быстрее
