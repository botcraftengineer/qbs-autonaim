## Как работает скоринг

Система скоринга QBS Автонайм присваивает каждому кандидату оценку от 0 до 100 баллов. Оценка показывает, насколько кандидат соответствует требованиям конкретной вакансии.

<Note>
  Один и тот же кандидат может иметь разные оценки для разных вакансий. Например, разработчик с 5-летним опытом получит 90 баллов на junior-позицию и 70 баллов на senior, если ему не хватает опыта лидерства.
</Note>

## Категории оценок

| Оценка | Категория | Рекомендация |
|--------|-----------|--------------|
| 80-100 | ⭐ Отличный | Приоритетный кандидат. Связаться в первую очередь |
| 60-79 | ✓ Хороший | Подходит. Рекомендуется рассмотреть |
| 40-59 | ○ Средний | Частично подходит. Рассмотреть при нехватке кандидатов |
| 0-39 | ✗ Не подходит | Не соответствует ключевым требованиям |

## Из чего складывается оценка

Итоговый балл формируется из нескольких компонентов:

### 1. Соответствие опыту (до 35 баллов)

```
• Точное совпадение с требуемым опытом: +35
• Опыт больше требуемого: +30-35
• Опыт немного меньше (до 1 года разницы): +20-25
• Опыт значительно меньше: +5-15
• Нет релевантного опыта: 0
```

### 2. Навыки и компетенции (до 35 баллов)

```
• Все обязательные навыки: +20
• Желательные навыки: +1-3 за каждый
• Бонусные навыки: +0.5-1 за каждый
• Смежные/связанные навыки: +0.5 за каждый
```

### 3. Образование и сертификаты (до 15 баллов)

```
• Профильное образование: +10
• Релевантные сертификаты: +1-2 за каждый
• Дополнительное обучение: +0.5-1 за курс
```

### 4. Дополнительные факторы (до 15 баллов)

```
• Локация (город/удалёнка): +5
• Зарплатные ожидания: +5
• Готовность к переезду/командировкам: +3
• Рекомендации: +2
```

## Влияние на воронку

Оценка влияет на рекомендуемые действия:

<AccordionGroup>
  <Accordion title="80-100 баллов — Автоматическое продвижение">

  </Accordion>
  
  <Accordion title="60-79 баллов — Уведомление рекрутеру">

  </Accordion>

  </Accordion>
  
  <Accordion title="Менее 40 баллов — Автоотказ">

  </Accordion>

## Калибровка скоринга

Если результаты скрининга не соответствуют вашим ожиданиям, вы можете откалибровать систему:

<Steps>
  <Step title="Оцените выборку вручную">
    Возьмите 10-15 кандидатов и оцените их вручную: «подходит» или «не подходит».
  </Step>
  <Step title="Сравните с AI">
    Посмотрите, совпадают ли ваши оценки с баллами AI.
  </Step>
  <Step title="Скорректируйте критерии">
    - Если хорошие кандидаты получают низкие баллы — смягчите требования
    - Если слабые кандидаты получают высокие баллы — ужесточите критерии
  </Step>
  <Step title="Измените веса">
    Перераспределите весовые коэффициенты, чтобы важные критерии влияли сильнее.
  </Step>
</Steps>

<Tip>
  Калибровку рекомендуется проводить после первых 50-100 откликов на вакансию, когда накопится достаточно данных.
</Tip>
