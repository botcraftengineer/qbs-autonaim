## Каналы уведомлений

QBS Автонайм отправляет уведомления через:

- **Email** — основной канал
- **Push в браузере** — мгновенные уведомления
- **Telegram** — через персонального бота
- **Slack/Teams** — интеграция с рабочими чатами

## Настройка по типам событий

### События по кандидатам

| Событие | По умолчанию | Настраиваемое |
|---------|--------------|---------------|
| Новый отклик | Email | ✅ |
| Высокий скор (80+) | Email + Push | ✅ |
| Кандидат застрял на этапе | Email | ✅ |
| Кандидат отказался | Email | ✅ |

### События по вакансиям

| Событие | По умолчанию | Настраиваемое |
|---------|--------------|---------------|
| Вакансия без откликов 7 дней | Email | ✅ |
| Достигнут лимит откликов | Email + Push | ✅ |
| Вакансия закрыта | Email | ✅ |

### Системные события

| Событие | По умолчанию | Настраиваемое |
|---------|--------------|---------------|
| Новый пользователь в команде | Email | ✅ |
| Проблема с интеграцией | Email + Push | ❌ |
| Обновления системы | Email | ✅ |

## Расписание уведомлений

Настройте время получения сводных уведомлений:

- **Утренняя сводка** — итоги за вчера (9:00)
- **Вечерняя сводка** — итоги за сегодня (18:00)
- **Еженедельный отчёт** — по понедельникам

## Тихие часы

Отключите уведомления в нерабочее время:

- Укажите рабочие часы (например, 9:00 – 19:00)
- Выберите рабочие дни (Пн-Пт)
- Срочные уведомления будут приходить всегда
