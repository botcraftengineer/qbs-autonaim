## Роли в системе

| Роль | Описание | Права |
|------|----------|-------|
| **Владелец** | Создатель аккаунта | Полный доступ, биллинг |
| **Администратор** | Управляющий | Всё, кроме биллинга |
| **Рекрутер** | Обычный пользователь | Работа с кандидатами и вакансиями |
| **Только чтение** | Наблюдатель | Просмотр без изменений |

## Приглашение коллег

<Steps>
  <Step title="Откройте настройки команды">
    **Настройки → Команда → Пригласить**
  </Step>
  <Step title="Введите email коллеги">
    Можно добавить несколько email через запятую.
  </Step>
  <Step title="Выберите роль">
    Укажите роль для новых пользователей.
  </Step>
  <Step title="Отправьте приглашение">
    Коллега получит письмо со ссылкой для активации.
  </Step>
</Steps>

<Info>
  Количество пользователей зависит от тарифа. На бесплатном тарифе — 1 пользователь.
</Info>

## Ограничение доступа

Вы можете ограничить доступ рекрутеров к определённым вакансиям:

### По вакансиям

- **Все вакансии** — видит всё
- **Только свои** — видит только назначенные ему вакансии
- **Выбранные** — видит конкретный список вакансий

### По данным

- **Зарплаты** — может/не может видеть зарплаты кандидатов
- **Контакты** — может/не может видеть телефоны и email
- **Комментарии коллег** — может/не может видеть внутренние заметки
