# Генератор вакансий с AI

Интерфейс для создания вакансий в стиле чата с использованием AI-ассистента.

## Особенности

- **Чат-интерфейс**: Диалог с AI для создания вакансии
- **Живое превью**: Вакансия отображается справа в реальном времени
- **Редактирование**: Возможность редактировать сгенерированную вакансию
- **Доступность**: Полная поддержка клавиатуры и screen readers
- **Адаптивность**: Работает на мобильных и десктопах

## Структура

```
generate/
├── page.tsx              # Главная страница генератора
└── README.md            # Документация
```

## Компоненты

### VacancyEditor
Компонент для редактирования вакансии:
- Редактирование названия, описания, зарплаты
- Добавление/удаление требований, обязанностей, условий
- Валидация полей

Расположение: `~/components/vacancy/vacancy-editor.tsx`

## Интеграция с AI

Для подключения реального AI:

1. Замените симуляцию в `handleSend()` на вызов API
2. Используйте существующий `/api/ai-proxy` endpoint
3. Обновляйте `vacancyData` на основе ответов AI

## Доступность

- Все интерактивные элементы имеют `aria-label`
- История чата помечена как `role="log"` с `aria-live="polite"`
- Поддержка навигации с клавиатуры
- Автофокус на input при загрузке

## Производительность

- Автоскролл к новым сообщениям
- Оптимизированные ре-рендеры
- Минимальные зависимости в useEffect
