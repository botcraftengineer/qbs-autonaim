# Demo Data Package

Пакет с демо данными для загрузки в базу данных системы QBS AutoNaim.

## Содержимое

### Вакансии (`data/vacancies.json`)
- **Senior Frontend Developer (React/TypeScript)** - вакансия с HH.ru
- **Python Backend Developer (Django/FastAPI)** - вакансия с HH.ru  
- **DevOps Engineer (Kubernetes/AWS)** - вакансия с SuperJob
- **Full Stack Developer (Node.js + React)** - e-commerce платформа
- **Mobile Developer (React Native)** - кроссплатформенная разработка
- **QA Engineer (Automation)** - автоматизация тестирования
- **Data Scientist (Python/ML)** - анализ данных и машинное обучение
- **UI/UX Designer** - дизайн интерфейсов
- **Java Backend Developer (Spring)** - enterprise системы
- **Product Manager (IT)** - управление IT продуктом

### Задания/Gigs (`data/gigs.json`)
- **Создание лендинга для стартапа** - верстка по Figma макету
- **Дизайн мобильного приложения для доставки** - полный UI/UX дизайн
- **Написание технических статей для блога** - копирайтинг
- **Настройка CI/CD для микросервисов** - DevOps автоматизация
- **Анализ данных продаж и создание дашборда** - аналитика
- **Создание промо-видео для продукта** - видеопродакшн
- **Перевод технической документации** - технический перевод
- **Консультация по архитектуре микросервисов** - консалтинг

### Отклики на вакансии (`data/responses.json`)
- 13 откликов от разных кандидатов
- Различные статусы: NEW, EVALUATED, SKIPPED
- HR статусы: RECOMMENDED, INVITE
- Scoring и ranking данные
- Контактная информация и навыки
- Ссылки на фото кандидатов

### Отклики на задания (`data/gig-responses.json`)
- 10 откликов на различные gigs
- Разнообразные типы заданий и кандидатов
- Полная информация о навыках и опыте
- Оценки и рекомендации
- Контактные данные и портфолио

### Фото кандидатов (`data/candidate-photos.json`)
- Профессиональные фото для каждого кандидата
- Ссылки на качественные изображения с Unsplash
- Автоматическая загрузка в систему файлов
- 10 различных кандидатов

### Интервью (`data/interview-sessions.json`, `data/interview-messages.json`)
- 4 интервью-сессии с AI-ботом
- 2 завершенных интервью через Telegram
- 1 завершенное интервью через Web
- 1 активное интервью (в процессе)
- 27 сообщений с реалистичными диалогами
- Вопросы и ответы кандидатов
- Метаданные с токенами и задержками

### Чаты (`data/chat-sessions.json`, `data/chat-messages.json`)
- 3 чат-сессии
- Командный чат по вакансии (рекрутер + менеджер)
- Чат заказчика с фрилансером по заданию
- AI-ассистент чат для работы с вакансией
- 8 сообщений с реальными диалогами
- Quick replies для AI-ассистента

## Использование

```bash
# Загрузить все демо данные (включая фото)
bun run load

# Или отдельно загрузить только фото
bun run upload-photos
```

## Структура данных

### Вакансии
- Полная информация о требованиях и навыках
- Настройки для бота и скрининга
- Статистика просмотров и откликов
- Источники: HH, SuperJob

### Задания (Gigs)
- Различные типы: DEVELOPMENT, DESIGN, COPYWRITING, VIDEO, TRANSLATION, CONSULTING
- Бюджет и сроки выполнения
- Детальные требования и deliverables
- Технический стек и уровень опыта
- Кастомные настройки для бота

### Отклики  
- Полиморфная структура (entityType: "vacancy" | "gig")
- Контактные данные кандидатов
- Технические навыки и опыт
- Scoring (0-100) и ranking позиции
- Анализ сильных/слабых сторон
- Рекомендации по найму
- **Фото кандидатов** через photoFileId

### Фото кандидатов
- Профессиональные портреты
- Загружаются в таблицу files
- Связываются с откликами через photoFileId
- Метаданные включают оригинальные URL

## Примечания

- Все данные являются демонстрационными
- UUID и ID будут сгенерированы автоматически при загрузке
- Скрипт автоматически связывает отклики с загруженными вакансиями и заданиями
- Фото загружаются из Unsplash и сохраняются в системе файлов
- Каждый кандидат получает профессиональное фото
- Поддерживаются как традиционные вакансии, так и разовые задания (gigs)
