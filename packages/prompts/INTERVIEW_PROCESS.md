# Процесс интервью через Telegram-бота

## Структура интервью

Интервью состоит из **2 этапов** (2 голосовых сообщения):

### Этап 1: Организационные вопросы
**Цель:** Узнать базовые организационные моменты
**Формат:** Одно голосовое сообщение от кандидата

**Вопросы (настраиваются в вакансии):**
- Какой график работы подходит?
- Какие ожидания по зарплате?
- Когда готовы приступить к работе?
- Рассматриваете ли релокацию?
- Есть ли другие офферы?

**Когда:** Сразу после ввода PIN-кода (этап PIN_RECEIVED) или в начале интервью (этап INTERVIEWING)

### Этап 2: Технические/профессиональные вопросы
**Цель:** Оценить профессиональный опыт и навыки
**Формат:** Одно голосовое сообщение от кандидата

**Вопросы (генерируются на основе вакансии):**
- Опыт работы с технологиями из требований
- Самый сложный/интересный проект
- Технологии последних 2 лет
- Почему заинтересовала позиция

**Когда:** После получения первого голосового (этап INTERVIEWING)

## Важные правила

1. **Краткость:** Бот задает все вопросы этапа в ОДНОМ сообщении (2-3 предложения)
2. **Не растягивать:** Не нужно задавать вопросы по одному - все сразу
3. **Два голосовых:** После получения 2 голосовых интервью завершается
4. **Текстовые ответы:** Если кандидат не хочет голосовые - принимаем текст
5. **Обращение:** Всегда на "Вы", без слова "Привет"

## Настройка

### Организационные вопросы
Настраиваются в UI вакансии:
- Поле: `customOrganizationalQuestions`
- Формат: Каждый вопрос с новой строки
- По умолчанию: график работы, зарплата, сроки начала, релокация

### Технические вопросы
Настраиваются в UI вакансии:
- Поле: `customInterviewQuestions`
- Формат: Каждый вопрос с новой строки
- По умолчанию: генерируются на основе требований вакансии

### Максимум голосовых
- Параметр: `maxVoiceMessages`
- По умолчанию: `2`
- Изменяется в коде (константа `DEFAULT_MAX_VOICE_MESSAGES`)

## Этапы диалога

```
1. AWAITING_PIN → Кандидат вводит PIN-код
2. PIN_RECEIVED → Приветствие + запрос 1-го голосового (организационные)
3. INTERVIEWING → Запрос 2-го голосового (технические) + завершение
4. ESCALATED → Передача живому рекрутеру (если нужно)
```

## Примеры сообщений бота

### Первое голосовое (организационные)
```
Добрый день! Запишите голосовое с ответами на пару вопросов: 
какой график работы вам подходит, какие ожидания по зарплате 
и когда готовы приступить?
```

### Второе голосовое (технические)
```
Отлично! Теперь запишите голосовое про опыт: расскажите про 
работу с React, самый интересный проект и почему заинтересовала 
эта позиция.
```

### Завершение интервью
```
Спасибо за ответы! Изучу информацию и свяжусь с вами 
в ближайшее время.
```
