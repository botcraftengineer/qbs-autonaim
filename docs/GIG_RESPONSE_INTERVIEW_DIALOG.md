# Отображение диалога с кандидатом на странице отклика

## Описание

Добавлена возможность просмотра истории диалога с кандидатом на странице детального просмотра отклика на гиг.

## Изменения

### API (`packages/api/src/routers/gig/responses/get.ts`)

- Добавлена загрузка `conversation` с сообщениями при получении отклика
- Фильтрация по `source = 'WEB'` для получения только веб-диалогов
- Сообщения сортируются по времени создания (от старых к новым)
- Включены связанные файлы для голосовых сообщений

### UI (`apps/app/src/components/gig/response-detail-card.tsx`)

- Добавлена новая вкладка "Диалог" в интерфейсе отклика
- Вкладка отображается только если есть сообщения в диалоге
- Сообщения отображаются в формате чата:
  - Сообщения бота слева с иконкой робота
  - Сообщения кандидата справа с иконкой пользователя
  - Поддержка голосовых сообщений с транскрипцией
  - Временные метки для каждого сообщения
- Используется `ScrollArea` для прокрутки длинных диалогов (высота 600px)

## Типы

```typescript
conversation?: {
  id: string;
  status: string;
  messages: Array<{
    id: string;
    sender: string;
    content: string;
    contentType: string;
    voiceTranscription: string | null;
    createdAt: Date;
  }>;
} | null;
```

## Доступность

- Семантическая разметка с использованием нативных компонентов
- Иконки с текстовыми метками для различения отправителей
- Контрастные цвета для сообщений бота и кандидата
- Поддержка клавиатурной навигации через вкладки
