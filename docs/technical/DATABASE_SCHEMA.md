# Схема базы данных

Полное описание структуры PostgreSQL базы данных платформы.

## Обзор

База данных построена на **PostgreSQL** с использованием **Drizzle ORM**. Схема поддерживает мультитенантность через workspaces и полиморфные связи для гибкости.

## Основные принципы

- **Мультитенантность** - изоляция данных по workspace
- **Полиморфизм** - единая таблица `responses` для разных типов откликов
- **Soft deletes** - мягкое удаление через поле `deletedAt`
- **Timestamps** - автоматические `createdAt` и `updatedAt`
- **UUID** - использование UUID для всех ID
- **JSONB** - гибкое хранение метаданных
