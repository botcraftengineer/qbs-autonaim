# Исправление: Организационные вопросы не задаются на gig-интервью

## Проблема

На gig-интервью (разовые задачи) бот задавал неуместные организационные вопросы:
- "Какой график работы вам подходит?"
- "Какие у вас ожидания по зарплате?"

Эти вопросы имеют смысл только для постоянных вакансий, но не для разовых задач (gig).

## Решение

### 1. Определение типа интервью

В `packages/jobs/src/services/interview/interview-service.ts` (строка 364-365):
- Для gig: `customOrganizationalQuestions: null`
- Для vacancy: `customOrganizationalQuestions: <значение из БД>`

### 2. Изменения в InterviewerAgent

В `packages/ai/src/agents/interviewer.ts`:

**Динамическая логика интервью:**
```typescript
const isGigInterview = input.customOrganizationalQuestions === null;
```

**Для gig-интервью:**
- Задаются ТОЛЬКО профессиональные/технические вопросы
- НЕ спрашивается про график, зарплату, формат работы
- Фокус на опыте, навыках, портфолио

**Для обычных вакансий:**
- Сначала организационные вопросы
- Потом технические вопросы

### 3. Изменения в InterviewStartAgent

В `packages/ai/src/agents/interview-start.ts`:
- Убраны жестко прописанные организационные вопросы из промпта
- Вопросы берутся только из `customOrganizationalQuestions` (если есть)
- Для gig (когда `customOrganizationalQuestions === null`) вопросы не показываются

## Результат

Теперь на gig-интервью бот задает только профессиональные вопросы:
- "Расскажите о вашем опыте в этой области?"
- "Какие похожие проекты вы делали?"
- "Что умеете делать лучше всего?"

А организационные вопросы задаются только для постоянных вакансий.
